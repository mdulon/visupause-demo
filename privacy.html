Politique de confidentialité

Dernière mise à jour : 11 juillet 2026

Résumé

VisuPause Pro est actuellement une application web statique. Elle ne crée pas de compte, n'utilise pas d'analytics, n'envoie pas de données applicatives vers un serveur et ne charge pas de service tiers.

Données traitées localement

L'application peut enregistrer localement dans ton navigateur les informations nécessaires au fonctionnement du produit : historique EyeScore, préférences de langue, paramètres d'exercices, mode d'affichage de pause et préférence de notifications. Ces données restent dans le localStorage de ton navigateur.

Notifications

Si tu actives les notifications, ton navigateur et ton système d'exploitation gèrent l'autorisation et l'affichage. VisuPause Pro utilise cette autorisation uniquement pour annoncer les pauses et la fin d'une pause. Tu peux désactiver les notifications dans l'application ou dans les réglages du navigateur/système.

PWA et cache hors ligne

Le service worker de l'application met en cache les fichiers nécessaires pour ouvrir l'application plus rapidement et permettre une utilisation hors ligne après une première visite. Ce cache ne contient pas de données de compte ni de données de paiement.

Cookies, analytics et partage

La version actuelle n'utilise pas de cookies, pas d'analytics, pas de publicité et pas de partage de données avec des tiers. Si une version payante, un hébergement avec authentification ou un outil de paiement est ajouté, cette politique devra être mise à jour avant le lancement public.

Contrôle de tes données

Tu peux effacer les données de VisuPause Pro en utilisant le bouton de réinitialisation de session dans l'application et/ou en supprimant les données du site dans les réglages de ton navigateur.

Contact

Avant un lancement public, ajoute ici une adresse de contact dédiée. Pour la beta privée, utilise le canal par lequel tu as reçu l'accès à l'application.

Privacy Policy

Last updated: July 11, 2026

Summary

VisuPause Pro is currently a static web application. It does not create accounts, use analytics, send app data to a server, or load third-party services.

Local data

The app may store locally in your browser the information needed to run the product: EyeScore history, language preference, exercise settings, break display mode, and notification preference. This data remains in your browser's localStorage.

Notifications

If you enable notifications, your browser and operating system handle permission and display. VisuPause Pro uses this permission only to announce breaks and break completion. You can disable notifications in the app or in browser/system settings.

PWA and offline cache

The app service worker caches the files required to open the app faster and allow offline use after a first visit. This cache does not contain account data or payment data.

Cookies, analytics, and sharing

The current version uses no cookies, no analytics, no advertising, and no third-party data sharing. If a paid version, authenticated hosting, or payment provider is added, this policy should be updated before public launch.

Your control

You can erase VisuPause Pro data by using the session reset in the app and/or deleting site data in your browser settings.

Contact

Before a public launch, add a dedicated contact address here. For the private beta, use the channel through which you received access to the app.
